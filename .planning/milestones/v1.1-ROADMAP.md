## Milestone Summary

**Key Decisions:**
- Next.js 16 (upgraded from planned 15.5) - sales website has no API routes
- Tailwind 4.0 CSS-first with @theme configuration
- Motion + GSAP two-library animation strategy for flexibility
- PostHog with reverse proxy (prevents 30-40% session loss from ad blockers)
- Orange primary color (#f97316) for conversion-optimized CTAs
- 69% certification accepted for MVP (solid technical foundation, iterate on content)

**Issues Resolved:**
- GSAP fromTo pattern for CSS opacity-0 elements
- Radix React 19 compatibility via rc versions
- OneDrive/pnpm EPERM issues (switched to npm)
- Provider nesting order for RTL + theme + analytics + motion

**Issues Deferred:**
- PROOF-07: Case study preview (awaiting real customer data)
- TRUST-06: Legal compliance pages (privacy policy, terms of service)

**Technical Debt Incurred:**
- Placeholder testimonial avatars (need real photos)
- Placeholder video testimonial file
- Placeholder Lottie animation and Storylane ID
- Contact phone/WhatsApp numbers are placeholders
- AnimatedButton exported but unused
