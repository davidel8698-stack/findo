Create worker for handling review reminder and auto-post logic:

1. **Worker setup:**
   - Listen on 'scheduled' queue
   - Filter to job.name === 'review-reminder'
   - Single concurrency

2. **sendReminders function:**
   - Query processedReviews where:
     * status = 'pending_approval'
     * approvalSentAt < now - 48 hours
     * reminderSentAt is NULL
   - For each review needing reminder:
     * Load tenant for ownerPhone
     * Create WhatsApp client via createWhatsAppClient(tenantId)
     * Check if session window is open using isWindowOpen from whatsapp/messages
     * Build reminder message in Hebrew:
       ```
       תזכורת: ממתינה תשובה לביקורת ({starRating} כוכבים)

       "{short comment preview, max 50 chars}..."

       תשובה מוצעת:
       "{draftReply}"
       ```
     * **If window open:** Use sendInteractiveButtons from whatsapp/messages:
       ```typescript
       await sendInteractiveButtons(
         client,
         tenant.ownerPhone,
         reminderMessage,
         [
           { id: `approve_${review.id}`, title: 'אשר ושלח' },
           { id: `edit_${review.id}`, title: 'רוצה לערוך' }
         ]
       );
       ```
     * **If window closed:** Use sendTextMessage with text instructions:
       "השב 'אשר' לאישור, או כתוב את התשובה שלך"
     * Update: status = 'reminded', reminderSentAt = now
     * Create activity event 'review.reminder_sent'
   - Rate limit: 100ms delay between tenants (use setTimeout)

3. **autoPostExpired function:**
   - Query processedReviews where:
     * status = 'reminded'
     * reminderSentAt < now - 48 hours
   - For each expired review:
     * Load googleConnection for postReviewReply
     * Post draftReply via postReviewReply(googleConnection, review.googleReviewId, review.draftReply)
     * Update: status = 'expired', postedReply = draftReply, repliedAt = now
     * Notify owner via sendTextMessage: "התשובה פורסמה אוטומטית לאחר 96 שעות"
     * Create activity event 'review.auto_posted'
   - Rate limit: 100ms delay between tenants

4. **Main worker processing:**
   ```typescript
   const { remindersCount } = await sendReminders();
   const { autoPostCount } = await autoPostExpired();
   console.log(`[review-reminder] Sent ${remindersCount} reminders, auto-posted ${autoPostCount}`);
   ```

5. **Error handling:**
   - Wrap each review processing in try/catch
   - Log errors but continue with other reviews
   - Don't fail job for individual failures
