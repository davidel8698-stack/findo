  <name>Task 1: Add gradient theme variables and text-gradient utility</name>
  <files>website/app/globals.css</files>
  <action>
Add gradient-related variables to the existing @theme block in globals.css:

1. Inside the existing `@theme { ... }` block, add these new variables:
```css
/* ============================================
 * GRADIENT TEXT SYSTEM - Phase 20
 * Premium gradient text for headlines
 * ============================================ */
--gradient-brand: linear-gradient(135deg, #f97316 40%, #f59e0b 60%);
--text-shadow-glow: 0 0 15px rgba(249, 115, 22, 0.35), 0 0 30px rgba(249, 115, 22, 0.2);
```

2. After the closing `}` of @theme block (but still in the file), add the functional utility:
```css
/* Gradient text utility - Phase 20 */
@utility text-gradient-* {
  @apply bg-clip-text text-transparent inline-block;
  background-image: --value(--gradient-*);
}
```

Key points:
- Use 135deg fixed angle (not directional keywords) for RTL consistency
- Color stops at 40%-60% for equal presence of orange and amber
- The glow shadow uses orange-500 color (#f97316) at low opacity
- `inline-block` prevents gradient clipping issues with descenders
  </action>
